# Cloud Jumper

A browser recreation of a childhood favorite — **Panfu's "Cloud Number Nine"** — reimagined and supercharged with new weapons, effects, and a dynamic day/night cycle to make the climb far more exciting than I remember it.

![Gameplay](short-gameplay.gif)

## Play online

👉 **[chahindb7.github.io/cloud-jumper](https://chahindb7.github.io/cloud-jumper/)**

## How to play

Steer with the mouse (or arrows / A / D). Keep bouncing up. Don't fall.

## Run it yourself

Docker:

```bash
docker compose up -d
```

Or just serve `index.html` with anything — even opening the file directly in a browser works.
